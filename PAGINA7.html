Conceptos de Lenguajes

Python
Lenguaje de alto nivel conocido por su sintaxis clara y legibilidad, muy usado en ciencia de datos.
JavaScript
Lenguaje esencial para el desarrollo web que permite crear contenido dinámico e interactivo.
C++
Lenguaje de propósito general que ofrece un alto rendimiento y control sobre los recursos del sistema.
PHP
Lenguaje de código abierto muy popular especialmente adecuado para el desarrollo web del lado del servidor.
